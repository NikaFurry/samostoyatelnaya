# Как внести вклад в проект
Спасибо за интерес к проекту! Вот как вы можете помочь:
## Клонирование репозитория
1. Сделайте форк репозитория.
2. Клонируйте ваш форк:
   ```bash
   git clone https://github.com/NikaFurry/samostoyatelnaya
   ```
## Отправка Pull Request
1. Создайте новую ветку: `git checkout -b feature/your-feature`.
2. Внесите изменения и закоммитьте: `git commit -m "Описание изменений"`.
3. Отправьте изменения: `git push origin feature/your-feature`.
4. Создайте Pull Request на GitHub.
## Правила оформления кода
- Следуйте PEP8 для Python.
- Пишите понятные комментарии.
- Тестируйте изменения перед отправкой.
